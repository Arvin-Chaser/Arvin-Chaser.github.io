在上一节中我们通过一个例子大概阐述了Primal LP与Dual LP的关系，本节中我们首先用更代数化的形式重写该过程。

Duality of Standard Form

给出一个标准形式的LP（mmm个约束条件，nnn个变量）
        minimize cTxc^TxcTx
        subject to Ax=bAx=bAx=b
        x≥0x≥0x≥0
我们可以将其重写为：
        minimize cTx+maxyT(b−Ax)c^Tx + max y^T(b-Ax)cTx+maxyT(b−Ax)，y∈Rmy\in R^my∈Rm
        subject to x≥0x≥0x≥0

  • 注意，通过在minminmin里引入maxyT(b−Ax)max y^T(b-Ax)maxyT(b−Ax)，我们隐含了Ax=bAx=bAx=b的条件——若Ax=bAx=bAx=b不得到满足，那么该最小化问题无法求解，因为其中的maxmaxmax可到达infinfinf

接下来，我们假定maxmaxmax和minminmin可以交换，那么此问题变为：
maxmaxmax bTy+minb^Ty + minbTy+min xT(c−ATy)x^T (c - A^Ty)xT(c−ATy)，其中x≥0x≥0x≥0

  • 注意，我们和前文相同，在这里优先分析minminmin的正负：
    • 当ATy≤cA^Ty≤cATy≤c时，minminmin为0
    • 当ATyA^TyATy不满足恒≤c≤c≤c时，minminmin为−inf-inf−inf
  • 所以为了使maxmaxmax问题有解，添加约束：ATy≤cA^Ty≤cATy≤c
  • 原问题变换为：
            maximize bTyb^TybTy
            subject toATy≤cA^Ty≤cATy≤c

这样的两个LP，我们就称它们是dual to each other的；若其中一个被我们定为Primal problem，那么另外一个就是Dual problem

  • 对于第一个最小化问题来说，yyy称为dual variables
  • 两个问题的最优值相等

Different Duality Forms

  • 我们将尝试推导一些非标准形式的二元性质

Problem 1

        minimize cTxc^TxcTx
        subject to Ax≥bAx≥bAx≥b

  • 同理，该方程可重写为：minminmin cTx+maxc^Tx + maxcTx+max yT(b−Ax)y^T(b-Ax)yT(b−Ax)，y≥0y≥0y≥0
  • 与前文相同，这里对yT(b−Ax)y^T(b-Ax)yT(b−Ax)求maxmaxmax的行为和该问题的有解性一起，保证了Ax≥bAx≥bAx≥b的约束条件成立
  • 我们再次复刻先前的操作，交换maxmaxmax和minminmin，得到
          maxmaxmax bTy+minb^Ty + minbTy+min xT(c−ATy)x^T(c-A^Ty)xT(c−ATy) ，y≥0y≥0y≥0
    进一步的，该方程等价于
        maximize bTyb^TybTy
        subject to ATy=cA^Ty=cATy=c
         y≥0y≥0y≥0
  • 注意：此处的约束条件是ATy=cA^Ty=cATy=c，而非不等关系，这是因为系数xTx^TxT没有规定正负，只要ATyA^TyATy和ccc不相等就会出现原问题无解的情况；对比之下，Primal Problem中我们提供的是不等式，是因为我们存在y≥0y≥0y≥0的约束条件

Duality Theory

Weak Duality Theorem

  • 回想上一节中我们对Primal LP和Dual LP函数值大小关系的辨析：
    • 假设Primal LP的objective function是maxmaxmax c1x1+...+cnxnc_1x_1+...+c_nx_nc1​x1​+...+cn​xn​，(x‾1,...x‾n)(\overline x_1, ... \overline x_n)(x1​,...xn​)是一个BFS
    • 假设Dual LP的objective function是minminmin b1y1+...+bmymb_1y_1+...+b_my_mb1​y1​+...+bm​ym​，(y‾1,...y‾m)(\overline y_1, ... \overline y_m)(y​1​,...y​m​)是一个BFS
    • 那么 c1x‾1+...+cnx‾n≤b1y‾1+...+bmy‾mc_1\overline x_1 + ... + c_n\overline x_n ≤ b_1\overline y_1 + ... + b_m\overline y_mc1​x1​+...+cn​xn​≤b1​y​1​+...+bm​y​m​

Strong Duality Theorem

  • 与weak duality theorem唯一的不同就是引入了两个确定的最优值：
    • 令(x1∗,...,xm∗)(x_1^*, ... ,x_m^*)(x1∗​,...,xm∗​)为primal LP的最优解
    • 令(y1∗,...,ym∗)(y_1^*, ... , y_m^*)(y1∗​,...,ym∗​)为dual LP的最优解
    • 那么c1x‾1+...+cnx‾n≤c1x1∗+...+cnxn∗=b1y1∗+...+bmym∗≤b1y‾1+...+bmy‾mc_1\overline x_1 + ... + c_n\overline x_n ≤ c_1x_1^* + ... + c_nx_n^* = b_1y_1^* + ... + b_my_m^* ≤ b_1\overline y_1 + ... + b_m\overline y_mc1​x1​+...+cn​xn​≤c1​x1∗​+...+cn​xn∗​=b1​y1∗​+...+bm​ym∗​≤b1​y​1​+...+bm​y​m​
  • 也就意味着：若一个primal 可行解和一个dual 可行解有着相同的函数值，那么它们分别为各自问题的最优解。

      1. The objective values coincide, i.e.i.e.i.e. cTx=bTyc^Tx = b^TycTx=bTy

  • 这也意味着，解决一个LP（假设是标准形式），也就等价于求解如下的线性系统：

    • Ax=bAx=bAx=b
    • ATy≤cA^Ty≤cATy≤c
    • bTy=cTxb^Ty=c^TxbTy=cTx

  • Strong Duality Theorem

    • 若一个LP存在一个最优解，那么它的Dual LP也同样存在一个最优解，且这两个最优值相同。
