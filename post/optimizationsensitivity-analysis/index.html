[Optimization]Sensitivity Analysis

https://Arvin-Chaser.github.io/post/optimizationsensitivity-analysis/
https://Arvin-Chaser.github.io
  • 在研究LP时，我们时常会考虑这种问题：
    • 当参数(A,b,c)(A,b,c)(A,b,c)改变时，最优解和最优值会如何变化？
  • 这种问题，我们称其为Sensitivity Analysis

Local Sensitivity

考虑一个标准形式的LP:
 maximize cTxc^TxcTx
  s.t. Ax=bAx=bAx=b
  x≥0x≥0x≥0

  • 我们将相应的最优值定为VVV

当AAA和ccc固定时

  • 当AAA和ccc固定时，VVV可被看作一个关于bbb的方程：V(b)V(b)V(b)

  • Theorem: Differentiability of V(b)V(b)V(b)

    • 若该问题的dual problem存在唯一的最优解y∗y^*y∗，那么∇bV(b)=y∗\nabla _bV(b) = y^*∇b​V(b)=y∗（注：该处的∇b\nabla _b∇b​是在微积分中提及过的概念，作为梯度向量，其每个分量为VVV对bbb各分量的偏导，即ith∇bV(b)=δVδbii^{th} \nabla _b V(b) = \frac{\delta V}{\delta b_i}ith∇b​V(b)=δbi​δV​；在该案例中，通俗的说，∇bV(b)\nabla _bV(b)∇b​V(b)描述了bbb的微小变化会如何影响最优值V(b)V(b)V(b)）
    • 若dual optimal solution不唯一（或un bdd.或不可行），那么梯度向量就不是完备定义的
    • 若bib_ibi​改变了一个微小量Δbi\Delta b_iΔbi​，那么objective value改变的微小量则是Δbiyi∗\Delta b_i y_i^*Δbi​yi∗​

  • 证明：

    • 由于primal problem的最优值VVV同样也是dual problem的最优值，所以对VVV的求解也可重写为： maximize bTyb^TybTy；s.t. ATy≤cA^Ty ≤ cATy≤c，代入y∗y^*y∗可知：V(b)=bTy∗V(b) = b^Ty^*V(b)=bTy∗
    • 写成这种类似线性的形式后，便可以直观体现，若bbb改变了一个微小量Δb\Delta bΔb，且最优解(y∗y^*y∗)不变，那么VVV的变化量为ΔbTy∗\Delta b^Ty^*ΔbTy∗
    • 注意，最优解的不变性在dual problem中得到体现，与下文中我们证明primal opt sol不变的过程相似

当AAA和bbb固定时

  • 此时VVV可被看作一个关于ccc的方程

  • Theorem: Differentiability of V(c)V(c)V(c)

    • 若原问题存在唯一的最优解x∗x^*x∗，那么∇cV(c)=x∗\nabla _c V(c) = x^*∇c​V(c)=x∗

  • 证明，V(c)=cTxV(c) = c^TxV(c)=cTx，当AAA和bbb确定时，解不变(x∗=A−1bx^* = A^{-1}bx∗=A−1b)，所以当ccc变化一个微小量Δc\Delta cΔc时，VVV的改变量必为∇cTx∗\nabla c^Tx^*∇cTx∗

当约束条件变为不等关系时

  • 先前的结论仍然成立，为证明这一点我们可以引入松弛变量后再将其转化为标准形式，之后沿用上面的分析过程

  • 首先讨论，在如下变量发生变动时，最优值会如何变化：
    • 1.bbb变化为b+Δbb+\Delta bb+Δb
    • 2.ccc变化为c+Δcc+\Delta cc+Δc
  • 回顾单纯形法中的表格：

  • 假设bbb变为了b~=b+Δb\tilde b = b + \Delta bb~=b+Δb，设原基为BBB，我们仍有：c‾=cT−cBTAB−1A≥0\overline c = c^T - c_B^TA_B^{-1}A ≥0c=cT−cBT​AB−1​A≥0，因为reduced cost是不随bbb的改变而改变的

    • 若我们想保证原来的基BBB对于新的问题仍是最优解，我们只需要验证新的基是否可行（因为最优性已由c‾>0\overline c >0c>0保证，我们只需要证明新的基仍可行，就可证明它仍是一组最优基）：若x~B=AB−1b~=AB−1(b+Δb)=xB∗+AB−1Δb≥0\tilde x_B = A_B^{-1}\tilde b = A_B^{-1}(b+\Delta b) = x_B^* + A_B^{-1}\Delta b ≥ 0x~B​=AB−1​b~=AB−1​(b+Δb)=xB∗​+AB−1​Δb≥0，那么BBB仍然是新问题的一组最优基

  • 若变化发生在一个基列中，那么基本上所有的变量都会受到影响，我们不能简单地得到所有变化的量及其范围，一般从头进行一次新的单纯形法，得到结果后一一对比

其它变化

添加一个变量

  • 若令新变量为000，则可令它为一个non-basic variable，那么原BFS仍是一个BFS，与原BFS相关的reduced costs没有变化
  • 我们只需要检验和新变量有关联的reduced costs
    • 若它非负，则原最优解加上新变量仍是最优解
    • 若它为负，从此处开始新一轮的单纯形法

添加一个约束条件

  • 若原最优解满足该约束条件，则原最优解仍是最优解
  • 若不是，则我们用一个dual variable来表示这个新的约束条件，然后以此为起点，对dual problem进行新一轮的单纯形法运算
